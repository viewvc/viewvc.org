Source Code History

ViewVC's source code history has been relocated and transformed a number of times.

  1. 1999-2001: Privately hosted CVS repository
  2. 2001-2005: CVS repository hosted at SourceForge
  3. 2005-2016: Subversion repository (via cvs2svn.py) hosted at Tigris.org
  4. 2016-present: Git repository (via GitHub Importer) hosted at GitHub
